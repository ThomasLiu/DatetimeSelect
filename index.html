Datetime Select

基于select的日期选择

引用

<!-- DatetimeSelect 核心 CSS 文件 -->
<link rel="stylesheet" href="css/main.css">

<!-- jQuery文件。务必在datetime_select.min.js 之前引入 -->
<script src="//cdn.bootcss.com/jquery/1.11.3/jquery.min.js"></script>

<!-- 最新的 DatetimeSelect 核心 JavaScript 文件 -->
<script src="js/datetime_select.min.js"></script>

bower

可以使用 bower 来直接下载，在项目中用命令行输入

bower install datetimeSelect --save

选择日期和时间

使用data-toggle="datetimeSelect"来注册控件

<input type="text" data-toggle="datetimeSelect">

只选择日期

使用data-toggle="dateSelect"来注册控件

<input type="text" data-toggle="dateSelect">

只选择时间

使用data-toggle="timeSelect"来注册控件

<input type="text" data-toggle="timeSelect">

调整最大和最小年份

年份选择默认 最大值 为当前年的 + 3 , 最小值 为当前年的 - 3 ，需要修改时可使用 data-maxyear 和 data-minyear来修改

<input type="text" data-maxyear="2035" data-minyear="1979" data-toggle="datetimeSelect">

默认当前时间

可使用 data-current 让控件初始化时为当前时间

<input type="text" data-current="true" data-toggle="datetimeSelect">

本来有值

可使用 value 让控件初始化时为该时间

<input type="text" value="2013/02/28 11:31" data-toggle="datetimeSelect">

必填

可使用 data-required 让控件必填

<input type="text" data-required="true" data-toggle="datetimeSelect">
